Теория формальных грамматик
Строгое математич описание языка
Алфавит - определены всякие теоретико-множественные операции, конкатенация символов
abc - конкатенация a b c
строки называются с большой буквы
* - замыкание
Алфавит в 1 степени - множество всех символов
        во 2 степени - множество цепочек из 2х символов
	в * степени - множество всех цепочек из любого количества символов
Язык - подмножество А*

Формальные грамматики - кортеж(терминальные символы T, нетерминальные символы N,
стартовый символ S, правила вывода R)
множества терминальных и нетерминальных символов не пересекаются
стартовые - подмножество нетерминальных

R подмножество (T U N)+ x (T U N)*
Пример 
S -> ab
Еще пример
1) Sa -> aS
2) S -> b

если применить 1 и 2 на строку Saa -> aSa -> aba
Если для данной цепочки символов существует вывод из стартового символа, то она
принадлежит языку

S -> BabSc
Bb -> b
Ba -> aB

Иерархия грамматик по Хомскому
type 0 - произвольная
type 1 - неукарачивающие грамматики ((T U N)* длиннее (T U N)+) - строка не 
уменьшается
type 2 - контекстно - свободные грамматики ((T U N)* заменяется на N)
type 3 - регулярные грамматики ((T U N)* - ровно 1 терминальный символ)
правило N x T x (N U пустой символ)

Тип 2 можно разобрать двумя алгоритмами 
1) Кука-Янгера-Косами (CYK)
2) Ёрли (Early) - поиск в ширину по деревьям разбора
просматриваем инпут слева направо и на каждом шаге строим множество всех возможных правил
которые могут учавствовать в разборе на iом символе

look-ahead - количество символов, на которые надо посмотреть вперед, чтобы разобрать грамматику
LL(k) - левый разбор
LR(k) - тоже левый разбор
